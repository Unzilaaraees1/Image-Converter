# 🖼️ Image Converter

A Python application that converts images into different formats such as PNG, JPG, JPEG, BMP, and WEBP.

## ✨ Features

- Convert PNG to JPG
- Convert JPG to PNG
- Support Multiple Image Formats
- Fast Image Processing
- Easy to Use

## 🛠 Technologies Used

- Python
- Pillow (PIL)

## 📂 Project Structure

Image-Converter/
│── main.py
│── sample_images/
│── output/
│── requirements.txt
│── README.md

## ▶️ How to Run

1. Install dependencies

```bash
pip install Pillow
```

2. Run

```bash
python main.py
```
